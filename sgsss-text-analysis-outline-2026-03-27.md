# Text Analysis for Social Scientists

## Event Summary

Computational methods are transforming research practice across the disciplines. In this one-day course you will learn how to apply computational methods for the analysis of text data. Using Python and R, you will develop skills in preprocessing text, creating document-term matrices, generating descriptive summaries, fitting topic models, and exploring how LLMs can support text analysis tasks.

## Event Description

In this one-day course you will learn how to apply computational methods for the analysis of text data. Using Python and R, you will develop skills in preprocessing text, creating document-term matrices, generating descriptive summaries (word clouds, keywords-in-context, collocations), fitting topic models (LDA), and exploring how LLMs can support text analysis tasks.

Learning outcomes:
1.	Understand how to use Python and R for quantitative text analysis in social science research.
2.	Ability to preprocess and represent text data for computational analysis.
3.	Apply descriptive and unsupervised methods to explore textual corpora.
4.	Critique the use of LLMs for analysing and annotating text data.
